## 分级学习体系

### 入门级（字母与拼读）
**学习目标**：掌握藏文字母系统，能够正确拼读
- 30个辅音字母 + 4个元音符号
- 上加字、下加字、前加字、后加字规则
- 叠字与音变规律
- 基础词汇1000词
- 日常简单对话

### 进阶级（语法与文体）
**学习目标**：掌握完整语法体系，能阅读一般藏文文本
- 八大格助词系统
- 动词四时态变位
- 敬语系统
- 书面语与口语差异
- 常用句型与句式
- 佛学术语2000词

### 高级（文献研读）
**学习目标**：能够阅读古典藏文文献与佛教论典
- 古藏文与中古藏文特点
- 不同宗派文献语言风格
- 五部大论选读
- 敦煌藏文文献阅读
- 藏文文体学

### 4个元音符号
- ི (i)：i 符，置于字母上方
- ུ (u)：u 符，置于字母下方
- ེ (e)：e 符，置于字母右上方
- ོ (o)：o 符，置于字母右上方

### 加字系统
1. **上加字**：ར་ལ་ས 可加于部分辅音字母上方
2. **下加字**：ཡ་ར་ལ་ཝ 可加于部分辅音字母下方
3. **前加字**：ག་ད་བ་མ་འ 可加于音节开头
4. **后加字**：ག་ང་ད་ན་བ་མ་འ་ར་ལ་ས 可加于音节末尾
5. **再后加字**：ད་ས 可加于部分后加字之后

## 语法体系

### 八大格助词系统

| 格 | 助词 | 功能 | 示例 |
|----|------|------|------|
| 第一格（体格） | 零形态 | 主语、主题 | བདག་（我） |
| 第二格（业格） | -ལ་/-ར་ | 宾语、方向 | བདག་ལ་（给我） |
| 第三格（具格） | -གིས་/-ཀྱིས་/-གྱིས་ | 工具、施动者 | བདག་གིས་（由我） |
| 第四格（为格） | -ལ་/-ར་/-ཏུ་ | 目的、对象 | ཆོས་ལ་（为法） |
| 第五格（从格） | -ནས་ | 来源、起点 | ཁྱིམ་ནས་（从家） |
| 第六格（属格） | -གི་/-ཀྱི་/-གྱི་ | 所属关系 | བདག་གི་（我的） |
| 第七格（依格） | -ན་ | 位置、时间 | ཁྱིམ་ན་（在家） |
| 第八格（呼格） | -ཧེ་/-ཧོ་ | 呼唤 | རྗེ་བཙུན་ཧེ་（尊者啊） |

### 动词时态系统
1. **现在时**：表示当前动作或常态
2. **过去时**：表示已完成的动作
3. **未来时**：表示将要发生的动作
4. **命令式**：表示祈使、命令

### 敬语系统
- **普通语**：日常对话使用
- **敬语**：对长辈、上师、尊者使用
- **最敬语**：对佛陀、上师使用最高敬语

## 文献阅读

### 推荐阅读进阶路径

#### 初级阅读材料
- 藏文童话与寓言
- 简单佛教故事
- 日常对话文本
- 现代藏文散文

#### 中级阅读材料
- 《菩提道次第广论》选读
- 《入菩萨行论》选读
- 藏传佛教祖师传记选读
- 现代藏文报刊文章

#### 高级阅读材料
- 五部大论原文
  1. 《释量论》（因明）
  2. 《现观庄严论》（般若）
  3. 《入中论》（中观）
  4. 《俱舍论》（俱舍）
  5. 《律经》（戒律）
- 敦煌藏文文献
- 各宗派重要论典

### 阅读技巧
1. **先通读后精读**：先了解大意，再逐句分析
2. **术语标注**：重要佛学术语标注威利转写与释义
3. **语法分析**：复杂句子先分析格助词与动词时态
4. **背景知识**：了解文本所属宗派与时代背景

## 学习工具生成

### 词汇闪卡格式
```
【藏文】བྱང་ཆུབ་སེམས་དཔའ་
【威利转写】byang chub sems dpa'
【词性】名词
【释义】菩萨，菩提萨埵
【例句】བྱང་ཆུབ་སེམས་དཔའ་རྣམས་ཀྱིས་སེམས་ཅན་ཐམས་ཅད་ལ་བརྩེ་བ་བྱེད།
【例句翻译】菩萨们对一切众生怀有慈悲心。
```

### 语法练习格式
**填空题**：
```
题目：ཁྱེད་རང་______ མིང་ག་རེ་རེད།
（你的名字是什么？）

选项：
A. གི་
B. གྱི་
C. ཀྱི་
D. གིས་

答案：C. ཀྱི་
解析："你的名字"中"你"（ཁྱེད་རང་）后接属格助词，以ང་结尾的词后接ཀྱི་。
```

### 翻译练习
**藏译汉**：提供句子，要求翻译成汉语
**汉译藏**：提供汉语句子，要求翻译成藏文
**段落翻译**：提供短文段落，进行完整翻译
