## 日本佛学研究概述

### 研究传统与特色
日本佛学研究历史悠久，形成了独特的学术传统：
1. **传统深厚**：自奈良时代起的佛教研究传统
2. **文献学扎实**：重视原典语言（梵文、藏文、汉文）研究
3. **部派研究细致**：各宗派都有专门研究机构
4. **国际交流活跃**：与欧美、亚洲各国学术交流频繁

### 主要研究领域
- 印度佛教（原始佛教、部派佛教、大乘佛教）
- 中国佛教（汉传佛教各宗派）
- 日本佛教（各宗派历史与教义）
- 藏传佛教（西藏、蒙古佛教）
- 敦煌学与西域佛教
- 佛教与现代社会

## 核心学术期刊

### 综合佛学期刊

1. **《印度学佛教学研究》**
   - 主办：日本印度学佛教学会
   - 地位：日本佛学研究最权威期刊
   - 特点：
     - 半年刊
     - 涵盖印度学、佛教学全领域
     - 论文质量高，审查严格
     - 日文为主，部分英文论文

2. **《日本佛教学会年报》**
   - 主办：日本佛教学会
   - 特点：年度论文集
   - 侧重：日本佛教研究

3. **《东洋学术研究》**
   - 主办：东洋大学
   - 特点：综合东洋学，含佛学
   - 历史悠久

4. **《大正大学综合佛教研究所年报》**
   - 主办：大正大学综合佛教研究所
   - 特点：
     - 综合佛教研究
     - 涵盖各领域

### 宗派研究期刊

1. **《禅文化研究所纪要》**
   - 主办：禅文化研究所
   - 特点：禅宗研究专业期刊
   - 侧重：临济宗、曹洞宗

2. **《密教文化》**
   - 主办：日本密教学会
   - 特点：密教研究专业期刊
   - 涵盖：真言宗、天台密教、藏传密教

3. **《净土学》**
   - 主办：净土宗学会
   - 特点：净土宗专门研究

4. **《天台学》**
   - 主办：天台学会
   - 特点：天台宗专门研究

5. **《华严学研究》**
   - 主办：华严学会
   - 特点：华严宗专门研究

### 藏学与西藏研究期刊

1. **《西藏研究》**
   - 主办：日本西藏学会
   - 特点：
     - 日本藏学研究核心期刊
     - 涵盖西藏历史、宗教、语言、文化
     - 年刊

2. **《日本西藏学会会报》**
   - 主办：日本西藏学会
   - 特点：学会会报，研究动态

### 国际期刊与英文出版物

1. **ARIRIAB（Annual Report of the International Research Institute for Advanced Buddhology）**
   - 主办：创价大学国际高等佛教研究所
   - 网址：http://iriab.soka.ac.jp/en/publication/aririab.html
   - 特点：
     - 英文年刊
     - 国际水准
     - 佛教研究各领域

2. **《佛教论坛》（The Buddhist Forum）**
   - 主办：国际佛教研究所（International Buddhist Studies College）
   - 网址：http://www.shin-ibs.edu/publications/the-buddhist-forum/
   - 特点：英文期刊

## 研究机构与大学

### 综合大学佛学研究

1. **京都大学**
   - 文学部 印度哲学佛教学专业
   - 人文科学研究所
   - 特点：日本佛学研究重镇，文献学传统深厚

2. **东京大学**
   - 文学部 印度哲学佛教学专业
   - 人文社会系研究科
   - 特点：综合实力强

3. **大正大学**
   - 综合佛教研究所
   - 特点：佛教专门大学，各宗派研究

4. **东洋大学**
   - 印度哲学佛教学科
   - 特点：历史悠久，佛教研究传统深厚

5. **创价大学**
   - 国际高等佛教研究所（IRIAB）
   - 特点：国际化程度高，英文出版多

6. **龙谷大学**
   - 佛教研究科
   - 特点：真宗系大学，敦煌学研究特色

7. **驹泽大学**
   - 禅研究所
   - 特点：曹洞宗系大学，禅宗研究强

8. **花园大学**
   - 禅学研究所
   - 特点：临济宗系大学，禅宗研究

### 专门研究机构

1. **国际佛教研究协会（IABS）日本支部**
   - 国际佛教研究协会日本分支

2. **佛教大学综合研究所**
   - 佛教大学附属研究机构

3. **禅文化研究所**
   - 京都
   - 禅宗文化研究

4. **密教研究所**
   - 高野山
   - 真言宗密教研究

## 数字资源与数据库

### 大藏经数据库

1. **大正新修大藏经（T）数字版**
   - 大正一切经刊行会
   - 最常用的汉文大藏经版本
   - 多种数字版本可用

2. **卍续藏经（X）数字版**
   - 续藏经数字化
   - 收录大量中国古代佛教著述

3. **南传大藏经数字版**
   - 巴利语大藏经
   - 日文译本数字版

### 文献数据库

1. **INBUDS（印度学佛教学术文献数据库）**
   - 日本印度学佛教学会运营
   - 收录日本发表的印度学佛教学论文
   - 论文摘要检索

2. **CiNii（学术论文数据库）**
   - 国立情报学研究所运营
   - 收录日本学术期刊论文
   - 佛学论文丰富

3. **J-STAGE**
   - 日本科学技术振兴机构运营
   - 电子期刊平台
   - 部分佛学期刊可开放获取

### 古籍数字资源

1. **京都大学人文科学研究所古籍数据库**
   - 收藏大量佛教古籍
   - 数字化程度高

2. **东京大学东洋文化研究所**
   - 汉籍数字资源
   - 含佛教文献

3. **龙谷大学数字图书馆**
   - 大谷探险队收集品
   - 敦煌、西域文献

## 藏传佛教与西藏研究

### 研究机构

1. **日本西藏学会**
   - 日本藏学研究核心组织
   - 会员涵盖各大学研究者

2. **京都大学西藏研究**
   - 文学部、人文研
   - 历史、宗教、语言研究

3. **东京外国语大学**
   - 藏语专业
   - 语言研究特色

4. **大阪大学**
   - 西藏佛教研究

### 研究特色
- 文献研究扎实：重视藏文原典解读
- 历史研究深入：西藏政治史、社会史
- 佛教研究精细：藏传佛教各宗派教义
- 田野调查传统：部分学者有西藏田野经验

### 重要学者（代表性）
- 山口瑞凤：西藏佛教史
- 佐藤长：西藏历史地理
- 羽田野伯猷：西藏佛教
- 松本史朗：西藏佛教思想史

## 敦煌学与西域研究

### 研究机构

1. **京都大学人文科学研究所**
   - 敦煌学研究传统深厚
   - 西域出土文献研究

2. **龙谷大学**
   - 大谷探险队收集品研究
   - 西域出土文献

3. **东京外国语大学**
   - 中亚语言研究
   - 粟特语、于阗语等

4. **早稻田大学**
   - 丝绸之路研究
   - 敦煌文学研究

### 研究特色
- 文献学功底深厚
- 语言研究发达（多种中亚古语言）
- 美术史研究有特色
- 与中国、欧洲学术交流密切

### 重要收藏
- **大谷探险队收集品**：龙谷大学、东京国立博物馆等
- **敦煌写本**：京都国立博物馆、东京国立博物馆等
- **西域美术品**：各大博物馆

## 学术出版与获取渠道

### 主要出版社

1. **岩波书店**
   - 学术出版重镇
   - 佛学著作多

2. **平凡社**
   - 东洋学出版知名
   - 佛教书籍丰富

3. **大藏出版**
   - 佛教专门出版社
   - 大藏经、佛书出版

4. **法藏馆**
   - 佛教专业出版社
   - 学术著作出版

5. **春秋社**
   - 宗教学术出版
   - 佛教书籍较多

### 丛书与系列

1. **《岩波讲座 东洋思想》**
   - 岩波书店
   - 含佛教各领域

2. **《大乘佛典》**
   - 岩波书店
   - 大乘经典日译与研究

3. **《佛教文化丛书》**
   - 大藏出版
   - 佛教研究专著系列

4. **《西藏佛教研究丛书》**
   - 藏学研究专著

### 论文获取方式
1. **J-STAGE**：部分期刊开放获取
2. **CiNii**：论文检索与部分全文
3. **INBUDS**：印度学佛教学术文献数据库
4. **研究机构repository**：各大学机构知识库
5. **直接联系作者**：日本学者通常乐于提供论文抽印本
