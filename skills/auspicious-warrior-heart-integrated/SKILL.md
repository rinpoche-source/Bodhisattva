# 吉祥勇士心・佛教研究 Skill 整合版

## 概述
「吉祥勇士心」是专为佛学研究者、藏语/梵语学习者、宗教学术从业者设计的专业 Skill，整合五大核心能力：藏文文献数字化（OCR + 翻译）、藏语系统学习、梵语语法与文献研读、全球佛学/藏学/敦煌学论文追踪、网站更新监控与学术雷达。

覆盖中、日、美、英、印、港台地区主流期刊，支持 Crossref 全网关键词监控、Google 学术快讯、知网/日本 CiNii RSS 抓取等高级追踪功能，整合敦煌学研究专区与藏学领域全链条资源。

## 核心工作流程

### 一、藏文文档翻译流程
1. **前置处理**：抽取文档前10页，清洗版权信息
2. **分批处理**：大文档自动切割，分批次翻译
3. **进度反馈**：每翻译完一个批次显示进度，中途不停止
4. **专业模式**：佛经等专业文档启用专家级翻译提示词，自动修正藏文拼写错误
5. **备用方案**：Token 不足时自动切换至 Dharmamitra 在线翻译，完成后合并交付

### 二、论文追踪与学术雷达
1. **多源订阅**：支持中、日、美、英、印、港台核心期刊 RSS 订阅
2. **全网监控**：Crossref DOI 关键词扫描，覆盖 Brill、Peeters 等小众出版商
3. **学术雷达**：Google Scholar 快讯转 RSS，追踪特定学者或主题
4. **知网抓取**：通过 RSSHub 抓取《中国藏学》《世界宗教研究》等核心期刊
5. **日文文献**：CiNii 关键词追踪，覆盖日本印度学佛教学最新成果
6. **本地记忆**：建立网站记忆库，对比更新，按"标题/作者/摘要"格式输出最新内容

### 三、网站更新跟踪
1. 用户可设置多个跟踪网站
2. 本地建立网站记忆库
3. 对比记忆库与网站最新内容
4. 按"标题、作者、摘要"形式输出更新

## 输出规范
- 藏文翻译标准三段式：原始藏文 → 威利转写 → 汉文译文
- 核心佛学术语加粗标注，异译与出处以脚注补充
- 论文追踪输出格式：标题、作者、摘要、DOI、获取链接
- 识别存疑字符统一用 `[?]` 标记，集中列于段落末尾

## 使用边界
- 本 Skill 为研究辅助工具，翻译结果仅供参考，重要文献需专业校核
- 学术资源链接可能变动，以实际访问为准
- 批量翻译需注意 Token 消耗，大文档建议分批处理
- RSSHub 等第三方服务稳定性依赖公共节点，必要时可自建实例
