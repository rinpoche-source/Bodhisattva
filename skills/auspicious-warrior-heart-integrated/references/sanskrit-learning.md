### 其他文字系统

1. **悉昙体（Siddham）**
   - 唐代传入中国的梵文字体
   - 日本密教仍在使用
   - 咒语书写常用

2. **兰札体（Ranjana）**
   - 尼泊尔、西藏使用的梵文字体
   - 藏传佛教梵文经典常用
   - 与天城体有对应关系

3. **IAST标准转写**
   - 国际通用梵语罗马化方案
   - 学术研究标准
   - 一一对应天城体字母

## 语法体系

### 八啭声（名词变格）

| 格 | 名称 | 主要用法 |
|----|------|----------|
| 第一格 | 体格（主格） | 主语、表语 |
| 第二格 | 业格（宾格） | 直接宾语、目的地 |
| 第三格 | 具格 | 工具、伴随、原因 |
| 第四格 | 为格（与格） | 目的、间接宾语 |
| 第五格 | 从格（来源格） | 来源、比较、原因 |
| 第六格 | 属格（所有格） | 所属关系 |
| 第七格 | 依格（位置格） | 地点、时间、条件 |
| 第八格 | 呼格 | 称呼 |

### 主要变格法
- **a-词干**：最常见，阳性、中性
- **ā-词干**：阴性名词
- **i-词干、u-词干**：包括阳性、阴性、中性
- **辅音词干**：较复杂，多种类型

### 十类动词变位

| 类别 | 名称 | 特点 | 示例 |
|------|------|------|------|
| 第一类 | 词根+a型 | 最常见 | bhū（成为） |
| 第二类 | 词根型 | 直接加词尾 | ad（吃） |
| 第三类 | 重叠型 | 词根重叠 | hu（祭祀） |
| 第四类 | ya型 | 词根+ya | div（游戏） |
| 第五类 | no型 | 词根+no | su（压榨） |
| 第六类 | a型（异于第一类） | 词根+a | tud（击打） |
| 第七类 | 中缀型 | 插入鼻音 | rudh（阻挡） |
| 第八类 | u型 | 词根+u | tan（伸展） |
| 第九类 | nā型 | 词根+nā | krī（购买） |
| 第十类 | aya型 | 词根+aya | cur（偷窃） |

### 主要时态与语气
- **现在时系统**：现在时、未然时、命令式、祈愿式
- **完成时**：重叠完成时
- **不定过去时**：多种类型
- **将来时**：sya型、iṣya型

### 复合词（Samasa）
- **依主释（Tatpuruṣa）**：前词限定后词
- **持业释（Karmadhāraya）**：前词形容后词
- **有财释（Bahuvrīhi）**：形容词性复合词
- **相违释（Dvandva）**：并列复合词
- **带数释（Dvigu）**：数词开头的复合词

### 连声（Sandhi）
- **词内连声**：词内部的音变
- **词间连声**：词与词之间的音变
- 元音连声、辅音连声、 Visarga 连声

## 佛典研读

### 佛教混合梵语（BHS）特点
- 语法简化：变格变位系统简化
- 俗语影响：受印度俗语（Prakrit）影响
- 词汇创新：佛教特有的词汇和用法
- 文本差异：不同部派经典语言有差异

### 主要佛典语言风格对比

#### 1. 阿含经/尼柯耶
- 较接近古典梵语
- 散文与偈颂结合
- 重复句式多
- 早期佛教语言

#### 2. 般若经类
- 佛教混合梵语特征明显
- 大量重复与排比
- 偈颂体为主
- 大乘佛教早期文献

#### 3. 律典
- 散文为主
- 法律文书风格
- 详细的条例表述
- 僧团规制语言

#### 4. 论典（阿毗达磨）
- 哲学术语密集
- 定义式表述
- 分类体系严密
- 学术性强

### 咒语（真言）研读
- 梵文原义解析
- 词源追溯
- 发音规则
- 密教咒语的特殊语法
