## 高级追踪功能

### 一、Crossref 全球出版物 DOI 关键词监控

利用 Crossref（全球最大的学术互联组织）配合 RSSHub 进行全网关键词扫描。一旦有包含 Buddhist 或 Tibetan text 的新书或新论文被注册 DOI，自动捕获。

#### 常用关键词 RSS 源

1. **佛教写本（Buddhist Manuscripts）**
   - URL: https://rsshub.app/crossref/search?query=Buddhist+Manuscripts

2. **密教研究（Tantric Buddhism）**
   - URL: https://rsshub.app/crossref/search?query=Tantric+Buddhism

3. **藏文文献（Tibetan Text）**
   - URL: https://rsshub.app/crossref/search?query=Tibetan+Text

4. **敦煌研究（Dunhuang Studies）**
   - URL: https://rsshub.app/crossref/search?query=Dunhuang

#### 特点
- 覆盖 Brill、Peeters 等小众但硬核的印藏学出版商
- 全网学术出版物实时监控
- 可自定义关键词组合

### 二、Google 学术自制雷达

由于 Google Scholar 取消了原生 RSS，可通过创建快讯（Alert）并利用电子邮箱转化为 RSS：

#### 设置步骤
1. 在 Google Scholar 检索关键词（如 author:"吉祥勇士心" 或 allintitle:"大伟的金刚"）
2. 点击左侧 "Create alert" 创建快讯
3. 将接收快讯的邮箱通过工具转化为专用 XML URL
4. 接入 RSS 阅读器进行统一订阅

#### 适用场景
- 追踪特定学者最新成果
- 追踪极细分主题（如特定经典、特定人物）
- 追踪特定机构研究动态

### 三、网站更新跟踪与本地记忆库

#### 功能说明
1. 用户可设置多个网站进行跟踪
2. 在本地建立网站记忆库信息
3. 每次查看网站最新信息时，对比本地记忆库
4. 将网站最新消息按"标题、作者、摘要"形式提供给用户

#### 输出格式
```
【网站名称】最新更新：
1. 标题：XXX
   作者：XXX
   摘要：XXX
   
2. 标题：XXX
   作者：XXX
   摘要：XXX
```

## RSSHub 代理路由

### 说明
知网和日本 CiNii 等没有对外的公开 Feed，通过 RSSHub 实例（或使用公共节点如 https://rsshub.app）通过模拟抓取将其封装为标准格式。

### 1. 中国知网 (CNKI) 核心期刊动态

知网路由协议为 `/cnki/journals/:name`（其中 :name 为期刊在知网的拼音/英文缩写代号）。

| 期刊名称 | RSSHub URL 地址 |
|----------|----------------|
| 《中国藏学》 | https://rsshub.app/cnki/journals/ZGZX |
| 《西藏研究》 | https://rsshub.app/cnki/journals/XZYJ |
| 《世界宗教研究》 | https://rsshub.app/cnki/journals/SJZJ |
| 《佛学研究》 | https://rsshub.app/cnki/journals/FXYJ |

### 2. 日本学术圈 (CiNii / INBUDS)

日本的《印度学仏教学研究》（INBUDS）和各大高校纪要一般通过 CiNii Research 索引。RSSHub 支持关键词检索生成 Feed。

| 关键词 | RSSHub URL 地址 | 用途 |
|--------|----------------|------|
| 仏教学 | https://rsshub.app/cinii/keyword/仏教学 | 追踪日本最新的佛教文献学论文 |
| サンスクリット | https://rsshub.app/cinii/keyword/サンスクリット | 追踪梵语/贝叶经新发现 |
| チベット | https://rsshub.app/cinii/keyword/チベット | 追踪日本藏学研究 |
| 敦煌 | https://rsshub.app/cinii/keyword/敦煌 | 追踪日本敦煌学研究 |

### 3. 使用建议
- 公共节点（rsshub.app）可能有限流，建议自建 RSSHub 实例
- 关键词可灵活组合，如 "仏教学 チベット"
- 定期检查 RSS 源有效性，部分路由可能随网站更新失效
